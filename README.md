讯业软件开发/讯业优众返利-java优众返利系统

### 一、产品概述
**“****优众返利****”是一款集****消费返现、社交裂变、商家入驻与分销代理****于一体的综合性****返利电商平台****，致力于打造“****用户省钱、商家增收、平台共赢****”的闭环商业生态。平台聚合餐饮、娱乐、旅游、购物、生活服务等多类线下商户资源，用户通过下单消费后，可获得一定比例的现金返利，实现“花钱还能赚钱”的新型消费体验。**

**该应用面向广大城市消费者、兼职创业者、社区团长及中小商户，提供便捷的返利查询、订单管理、收益提现、团队招募等功能，支持个人发展分销网络，成为一级或高级代理，实现“边消费边赚钱”的灵活创收模式。同时，平台为商家提供低成本引流、精准营销和客户沉淀工具，助力其拓展客源、提升转化率。**

**“优众返利”不仅是消费返利工具，更是一个融合社交电商、本地生活服务与轻创业机会的数字商业平台，推动传统消费向“价值共享”模式升级。**

### 二、核心功能
1. **分类****  
****按行业（如餐饮、酒店、KTV、美容、洗浴、教育、旅游等）展示合作商家，支持快速筛选与浏览。**
2. **我要赚钱****  
****进入收益中心，查看当前可赚取的返利金额、任务奖励、推广收益等，激发用户参与积极性。**
3. **商家入驻****  
****提供商家注册入口，支持本地实体商户申请加入平台，提交资质后审核上线，获取流量支持。**
4. **收藏****  
****收藏常用商家或优惠活动，方便下次快速访问与下单。**
5. **我的团队****  
****查看自己发展的下级代理名单，管理团队成员，实时掌握团队业绩与收益分成情况。**
6. **客服****  
****提供在线客服通道，解答用户关于返利规则、订单问题、提现异常等疑问，保障服务质量。**
7. **投诉建议****  
****用户可提交对商家服务、平台功能的意见反馈，平台及时处理并优化体验。**
8. **返利设置****  
****设置自动返利规则、收益提醒方式、收款账户等个性化选项，提升使用便捷性。**
9. **我的订单****  
****查看所有已下单的消费记录，包括待核销、已完成、已返利等状态，支持订单详情查询与申诉。**
10. **提现****  
****将累计收益转入微信、支付宝或银行卡账户，支持实时到账，操作简单安全。**
11. **收益码****  
****生成专属二维码，用于分享给好友或张贴在门店，他人扫码下单即可产生返利，实现被动收入。**
12. **商家端****  
****专为入驻商家设计的功能入口，支持查看订单、管理优惠券、发布活动、查看数据分析等。**
13. **销售端****  
****面向代理商或地推人员，提供业绩统计、客户管理、任务派发等功能，便于开展业务。**
14. **一级代理****  
****用户可通过邀请新人注册并消费，获得佣金分成，成为一级代理，开启轻创业之路。**
15. **高级代理****  
****达成一定条件后可升级为高级代理，享受更高返佣比例、专属客服、团队管理权限等特权。**
16. **首页推荐****  
****（主界面功能）展示热门商家、限时优惠、节日福利、返利排行榜等，增强用户活跃度。**
17. **搜索与定位****  
****支持按关键词搜索商家，或基于地理位置推荐周边可用服务，提升使用效率。**
18. **返利比例展示****  
****每个商家页面明确标注返利比例（如1%-20%），透明公开，增强用户信任。**
19. **每日签到与任务奖励****  
****（隐含功能）用户每日签到、完成指定任务可获得额外积分或现金奖励，提升留存率。**
20. **活动中心****  
****发布节日促销、新店开业、满减优惠等活动信息，吸引用户参与消费。**

### 三、产品价值
#### **用户价值：让每一分钱都花得更有回报**
+ **省钱又赚钱****：消费即返现，最高可达20%，真正实现“买得越多，赚得越多”，降低生活成本。**
+ **轻松创收****：无需投入本金，通过分享链接、发展团队即可获得持续收益，适合兼职、宝妈、学生等群体。**
+ **本地生活便利****：整合身边优质商家资源，一键预约、扫码核销，简化消费流程。**
+ **社交裂变激励****：通过“收益码”“团队分红”机制，鼓励用户主动传播，形成自增长闭环。**
+ **数据透明可控：所有返利记录、提现明细、团队业绩均可查，确保公平公正。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

+ **2 核 2 G 100MB 3M带宽 为最低配置，建议 2 核 4G 100Mb 5M带宽。**
+ **Node.js 14x**
+ **Java1.8**
+ **Mysql 5.7**
+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
